# 职能图谱

面向高校学生的企业岗位能力画像网站。当前覆盖立芯科技、上海超级计算中心与米哈游 7 个代表性技术岗位，提供岗位画像卡、8 维能力矩阵，以及课程、竞赛、项目和职业发展建议。

## 简历工坊

页面内置三种简历表达模板，并支持以下流程：

1. 从岗位画像直接带入目标岗位，或在简历工坊中自行选择。
2. 一键载入虚构范例简历，参考信息顺序与项目表达方式。
3. 填写教育、技能、项目、实践和成果信息。
4. 从下拉菜单选择 OpenAI、Gemini、DeepSeek、通义千问、Kimi 或智谱 GLM，也可填写自定义 OpenAI 兼容服务。
5. 使用 AI 生成岗位定制简历，或先生成完全不联网的本地草稿。
6. 复制简历文本、下载独立 HTML 文件，或通过浏览器打印为 PDF。

API 密钥只保留在当前页面内存中，不会写入 `localStorage`。服务商、接口地址和模型名称会作为非敏感偏好保存在浏览器本地。由于请求由浏览器直接发起，AI 服务需要允许当前站点进行跨域请求。正式部署时，如果服务商不支持浏览器跨域或需要隐藏密钥，建议增加服务端代理。

## 本地预览

直接打开 `index.html`，或在本目录运行：

```powershell
python -m http.server 4173
```

然后访问 `http://localhost:4173`。

## 部署到 GitHub Pages

1. 在 GitHub 新建公开仓库，例如 `talent-atlas`。
2. 将本目录中的文件提交并推送到仓库默认分支 `main`。
3. 进入仓库 `Settings > Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/(root)`。
5. 保存后等待 Pages 发布，地址通常为 `https://<用户名>.github.io/talent-atlas/`。

本项目使用相对路径，可直接部署在 GitHub Pages 的项目子路径中，无需构建步骤。

## 研究声明

岗位能力分值由公开招聘文本归纳而来，用于学习规划与研学展示，不代表企业官方评价或实际录用门槛。求职前请再次核对企业招聘官网。
